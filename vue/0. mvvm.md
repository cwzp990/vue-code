**使用 jQuery 和 Vue 的区别？**

![mock-pic2](https://github.com/cwzp990/vue-code/tree/master/vue/images/jQuery.png)

jQuery:数据视图没有分离，违背了开放-封闭原则(解耦)

Vue:以数据驱动视图，只更改数据，而不关心视图(DOM)

**说一下对 MVVM 的理解**

MVC: modle -> view -> controller

MVVM: view(DOM) -> ViewModel(Vue) -> model(DATA)
